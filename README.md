# Team_18 – College Bus Tracking System

## Project Description
The College Bus Tracking System is a full-stack web application developed to help students, drivers, and administrators track college buses in real time. It improves transportation management by providing live bus locations, estimated arrival times, route details, and user-friendly dashboards.

This project is designed to reduce waiting time, improve communication, and increase safety and convenience for students using college transport services.

## Features
- User authentication (Login / Register)
- Student dashboard
- Admin dashboard
- Real-time bus tracking
- Route management
- Driver management
- ETA (Estimated Time of Arrival)
- Notifications / Alerts
- Secure backend APIs
- Responsive frontend design

## Tech Stack
### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
